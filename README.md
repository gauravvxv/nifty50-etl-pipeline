# Nifty 50 ETL pipeline

Investors and analysts often struggle to track NIFTY 50 stocks manually across multiple companies and sectors. This project builds an ETL pipeline that automatically collects stock data, transforms it into analysis-ready format, and stores it for reporting and dashboarding.

The goal is to reduce manual effort in collecting stock data and create a reliable reporting layer for analyzing stock performance, volatility, returns, and sector-wise trends.
